Cross-Browser Web SDK Documentation

Owner: Okoye Chibuike

1. Introduction

Cross-Browser is a lightweight, intent-aware Web SDK designed to modernize browser interaction. By decoupling UI input from native browser constraints, it provides developers with a high-performance, programmable runtime environment. It empowers developers to reclaim control over browser-level interactions, such as touch gestures and menu systems, across any device.

2. Core Architecture

The SDK operates through a dual-layer architecture:

  • Main Thread (Interaction Layer): Handles global event delegation, feature detection for device-specific inputs (touch vs. click), and DOM manipulation.
  • Worker Thread (Synthesis Engine): Processes intent synthesis (dwell duration, gesture recognition) in the background to ensure a fluid, jank-free user experience.

3. Quickstart Integration

Ensure you have the following two files in your project root: intent.html and worker.js.

Full Implementation Code

intent.html

<!DOCTYPE html>
<html>
<head>
    <style>
        body { margin: 0; width: 100vw; height: 100vh; -webkit-user-select: none; user-select: none; -webkit-touch-callout: none; overflow-x: hidden; font-family: sans-serif; }
        #content-text { font-size: 16px; padding: 20px; min-height: 100px; -webkit-user-select: text; user-select: text; border: none; margin: 0; }
        #action-menu { display: none; position: fixed; top: 50%; left: 50%; transform: translate(-50%, -50%); background: #f9f9f9; border: 1px solid #ddd; padding: 15px; border-radius: 12px; box-shadow: 0 10px 25px rgba(0,0,0,0.3); z-index: 1000; width: 85%; max-width: 320px; }
        .menu-grid { display: grid; grid-template-columns: 1fr 1fr; gap: 8px; margin-top: 10px; }
        button { padding: 10px; border: none; background: #e0e0e0; border-radius: 5px; font-size: 14px; }
        .close-btn { width: 100%; margin-top: 10px; background: #ff5e5e; color: white; }
    </style>
</head>
<body>
    <div id="content-text" contenteditable="true"></div>

    <div id="action-menu">
        <h3>Browser Actions</h3>
        <div class="menu-grid">
            <button onclick="performAction('COPY')">Copy</button>
            <button onclick="performAction('PASTE')">Paste</button>
            <button onclick="performAction('SHARE')">Share</button>
            <button onclick="performAction('REFRESH')">Refresh</button>
            <button onclick="performAction('SELECT_ALL')">Select All</button>
            <button onclick="performAction('EXPORT_STATE')">Export State</button>
        </div>
        <h3>Formatting</h3>
        <div class="menu-grid">
            <button onclick="performAction('BOLD')">Bold</button>
            <button onclick="performAction('ITALIC')">Italic</button>
            <button onclick="performAction('COLOR')">Text Color</button>
            <button onclick="performAction('SIZE')">Font Size</button>
        </div>
        <button class="close-btn" onclick="closeMenu()">Close</button>
    </div>

    <script>
        const isTouchDevice = 'ontouchstart' in window || navigator.maxTouchPoints > 0;
        const worker = new Worker('worker.js');
        let isHolding = false;
        let touchStartTime = 0;

        document.addEventListener('selectstart', (e) => { if (e.target.id !== 'content-text') e.preventDefault(); }, false);
        document.addEventListener('contextmenu', (e) => e.preventDefault(), false);

        if (isTouchDevice) {
            document.addEventListener('touchstart', (e) => { if (e.target.id !== 'content-text') { isHolding = true; touchStartTime = Date.now(); } }, { passive: false });
            document.addEventListener('touchend', () => { isHolding = false; }, { passive: false });
        } else {
            document.addEventListener('mousedown', (e) => { if (e.target.id !== 'content-text') { isHolding = true; touchStartTime = Date.now(); } });
            document.addEventListener('mouseup', () => { isHolding = false; });
        }

        setInterval(() => {
            if (isHolding) { worker.postMessage({ type: 'CAPTURE_INTENT', data: { dwellDuration: Date.now() - touchStartTime, target: 'global-body' } }); }
        }, 500);

        worker.onmessage = (e) => { if (e.data.type === 'APPLY_LAYOUT' && e.data.layout.action === 'OPEN_MENU') { document.getElementById('action-menu').style.display = 'block'; } };
        function closeMenu() { document.getElementById('action-menu').style.display = 'none'; }

        async function performAction(type) {
            const content = document.getElementById('content-text'); content.focus();
            switch(type) {
                case 'COPY': document.execCommand('copy'); break;
                case 'PASTE': try { const text = await navigator.clipboard.readText(); document.execCommand('insertText', false, text); } catch (err) { alert("Paste permission denied."); } break;
                case 'SELECT_ALL': document.execCommand('selectAll', false, null); break;
                case 'REFRESH': location.reload(); break;
                case 'SHARE': if (navigator.share) navigator.share({title: document.title, url: window.location.href}); break;
                case 'BOLD': document.execCommand('bold', false, null); break;
                case 'ITALIC': document.execCommand('italic', false, null); break;
                case 'COLOR': const color = prompt("Color:", "red"); if (color) document.execCommand('foreColor', false, color); break;
                case 'SIZE': const size = prompt("Size (1-7):", "5"); if (size) document.execCommand('fontSize', false, size); break;
                case 'EXPORT_STATE': const blob = new Blob([content.innerText], { type: 'text/plain' }); const url = URL.createObjectURL(blob); const a = document.createElement('a'); a.href = url; a.download = 'runtime-state.txt'; a.click(); URL.revokeObjectURL(url); break;
            }
            closeMenu();
        }
    </script>
</body>
</html>

worker.js

// worker.js - Synthesis Engine for Cross-Browser SDK
// Owner: Okoye Chibuike

self.onmessage = function(e) {
    if (e.data.type === 'CAPTURE_INTENT') {
        const { dwellDuration } = e.data.data;
        // Define dwell threshold (800ms = 0.8 seconds)
        const DWELL_THRESHOLD = 800; 
        if (dwellDuration >= DWELL_THRESHOLD) {
            self.postMessage({ type: 'APPLY_LAYOUT', layout: { action: 'OPEN_MENU' } });
        }
    }
};

4. API Reference

The SDK automatically detects if the user is on a touch device or desktop via isTouchDevice and binds the appropriate listeners (Touch vs. Mouse). Developers can extend the performAction(type) function to include custom business logic or unique application commands.

5. Developer Customization

The SDK acts as a non-destructive overlay. It captures events globally but ignores interactions with specified editable containers, allowing developers to maintain existing input functionality while the rest of the application remains under the control of the Cross-Browser runtime.
