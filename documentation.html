Developer Documentation & Edge API Integration Guide

1. Publisher Integration (Web & WebViews)

To display Connectgold AI-optimized ads on your website or mobile WebView, insert the code snippet below into your page markup. Requests are served via our AI Gateway edge engine for maximum speed and CTR optimization.

Standard Banner Unit (320x50)

<!-- Connectgold Ad Network Unit -->
<div id="plc_banner_unit" style="width:320px; height:50px; margin:auto;"></div>
<script>
  (function() {
    function loadAd() {
      fetch("https://broad-brook-39f4.trustconnect713.workers.dev/api/ad?plc=plc_banner_unit&pub_id=YOUR_PUBLISHER_ID&format=banner&consent=1")
        .then(res => res.json())
        .then(ad => {
          if (!ad) return;
          const box = document.getElementById("plc_banner_unit");
          box.innerHTML = ad.adHtml || `<a href="${ad.click_url}" target="_blank"><img src="${ad.payload.image_url}" width="320" height="50" /></a>`;
          if (ad.impression_url) fetch(ad.impression_url);
        });
    }
    loadAd();
    setInterval(loadAd, 30000); // Auto-refresh auction every 30 seconds
  })();
</script>
