Connectgold Ads
Simple Developer Documentation & SDK Integration Guide

1. Publisher Integration (Web & WebViews)

To display Connectgold banner ads on your website or mobile WebView, add the code snippet below where you want the ad to appear.

Standard Banner (320x50)

<!-- Connectgold Banner Unit -->
<div id="cg_banner_unit" style="width:320px; height:50px; margin:auto;"></div>
<script>
  (function() {
    function loadAd() {
      fetch("https://lingering-sound-9859.trustconnect713.workers.dev/v1/ad?placement_id=cg_banner_unit&format=banner&consent=1")
        .then(res => res.json())
        .then(ad => {
          if (!ad || !ad.payload) return;
          const box = document.getElementById("cg_banner_unit");
          box.innerHTML = `<a href="${ad.click_url}" target="_blank"><img src="${ad.payload.image_url}" width="320" height="50" /></a>`;
          if (ad.impression_url) fetch(ad.impression_url);
        });
    }
    loadAd();
    setInterval(loadAd, 30000); // Auto-refresh every 30 seconds
  })();
</script>

2. Authorized Digital Sellers (ads.txt)

Add this single line to your root domain's ads.txt file (e.g. https://yourdomain.com/ads.txt) to verify your domain:

lingering-sound-9859.trustconnect713.workers.dev, YOUR_PUBLISHER_ID, DIRECT, f08c47fec0942fa0

3. Advertiser Campaign API

Advertisers can publish campaigns directly into the real-time bidding auction engine.

Endpoint: POST https://lingering-sound-9859.trustconnect713.workers.dev/v1/advertiser/add-ad

Request Body (JSON)

{
  "id": "ad_01",
  "campaign_id": "camp_01",
  "format": "banner",
  "title": "My First Campaign",
  "destination_url": "https://example.com",
  "image_url": "https://example.com/banner.png",
  "bid_rate": 2.50,
  "budget": 50.00
}

4. Analytics Reporting API

Fetch daily stats dynamically using the Analytics Worker endpoint.

Endpoint: GET https://floral-pine-5c44.trustconnect713.workers.dev/v1/analytics?format=banner&date=YYYY-MM-DD

Sample Response

{
  "date": "2026-07-21",
  "format": "banner",
  "impressions": 21,
  "clicks": 3,
  "ctr": "14.29%"
}
