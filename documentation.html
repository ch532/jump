Connectgold Ads
Developer Documentation & Edge API Integration Guide

1. Architecture Overview

Connectgold Edge Ad Network delivers AI-optimized programmatic ads directly at Cloudflare edge nodes. Real-time contextual decisions are routed via Cloudflare Workers and Workers AI to maximize yield and minimize latency for web platforms and mobile WebViews.

Core Request Parameters

Parameter Type Description
type String Ad unit format: banner, native, interstitial
plc String Unique placement ID on your domain/app (e.g., plc_banner_300)
pub_id String Your verified Publisher Account ID
consent String User GDPR/CCPA consent state (1 = Granted, 0 = Denied)

2. Publisher Integration Snippets

A. Banner Ad Unit (300x250)

Place this container and script in your page HTML where you want the visual banner unit to display.

<!-- Connectgold Banner Unit -->
<div id="plc_banner_300" style="width:300px; min-height:250px; margin:20px auto; overflow:hidden;"></div>
<script>
  (function() {
    function fetchAndRenderAd() {
      const consentVal = "1";
      fetch("https://broad-brook-39f4.trustconnect713.workers.dev/ad-request?type=banner&plc=plc_banner_300&pub_id=YOUR_PUBLISHER_ID&consent=" + consentVal)
        .then(res => res.json())
        .then(data => {
          if (!data || !data.success) return;
          const box = document.getElementById("plc_banner_300");
          if (!box) return;
          
          if (data.adHtml || data.html_snippet) {
            const tempDiv = document.createElement("div");
            tempDiv.innerHTML = data.adHtml || data.html_snippet;
            box.innerHTML = "";
            Array.from(tempDiv.childNodes).forEach(node => {
              if (node.tagName === "SCRIPT") {
                const newScript = document.createElement("script");
                Array.from(node.attributes).forEach(attr => newScript.setAttribute(attr.name, attr.value));
                if (node.innerHTML) newScript.appendChild(document.createTextNode(node.innerHTML));
                box.appendChild(newScript);
              } else {
                box.appendChild(node.cloneNode(true));
              }
            });
          } else if (data.ad) {
            const adInfo = data.ad;
            box.innerHTML = `<a href="${adInfo.destination_url}" target="_blank" rel="noopener">
              <img src="${adInfo.image_url}" width="300" height="250" alt="${adInfo.title}" style="border:0; max-width:100%; height:auto; border-radius:6px;" />
            </a>`;
            if (adInfo.nurl) fetch(adInfo.nurl);
          }
        })
        .catch(err => console.error("Connectgold Ad Error:", err));
    }
    fetchAndRenderAd();
    setInterval(fetchAndRenderAd, 30000); // 30-second auction auto-refresh
  })();
</script>

<script>
  function closeInterstitialAd() {
    document.getElementById("cg-interstitial-modal").style.display = "none";
  }

  function triggerAutoInterstitial() {
    const consentVal = "1";
    fetch("https://broad-brook-39f4.trustconnect713.workers.dev/ad-request?type=interstitial&plc=plc_interstitial&pub_id=YOUR_PUBLISHER_ID&consent=" + consentVal)
      .then(res => res.json())

        if (!data || !data.success || !data.ad) return;
        const adInfo = data.ad;
        document.getElementById("cg-int-title").innerText = adInfo.title || "Featured";
        document.getElementById("cg-int-desc").innerText = adInfo.description || "";
        document.getElementById("cg-int-img").src = adInfo.image_url || "";
        document.getElementById("cg-int-cta").href = adInfo.destination_url;
        document.getElementById("cg-int-cta").innerText = adInfo.cta_text || "Learn More";

  // Trigger automatically 3 seconds after page load
  window.addEventListener("DOMContentLoaded", () => {
    setTimeout(triggerAutoInterstitial, 3000);
  });
</script>

3. Multi-Engine Analytics & Telemetry

Connectgold ad units feature zero-configuration telemetry. Ad impression callbacks automatically detect and report impression events to active third-party analytics engines loaded on the host page:

  • Google Analytics (GA4): Dispatches via gtag('event', 'connectgold_ad_impression', ...)
  • Mixpanel: Tracks event via mixpanel.track('connectgold_ad_impression', ...)
  • Plausible Analytics: Logs privacy-first goal via plausible('connectgold_ad_impression', { props: {...} })

Sample Unified Impression Callback

function trackConnectgoldImpression(adData) {
  const eventName = 'connectgold_ad_impression';
  const payload = {
    ad_id: adData.ad ? adData.ad.campaign_id : 'house',
    placement_id: adData.plc || 'plc_demo',
    format: adData.type || 'banner'
  };

  // 1. Google Analytics 4
  if (typeof gtag === 'function') {
    gtag('event', eventName, payload);
  }

  // 2. Mixpanel
  if (typeof mixpanel === 'object' && typeof mixpanel.track === 'function') {
    mixpanel.track(eventName, payload);
  }

  // 3. Plausible Analytics
  if (typeof plausible === 'function') {
    plausible(eventName, { props: payload });
  }
}

4. Integration Best Practices for Developers

  • DOM Placement: Always place Banner and Native tags inside the <body> element where the graphic should physically display. Never place visual unit containers inside <head>.
  • Unique Placement IDs: Ensure each ad unit on a page uses a unique container ID and plc parameter (e.g., plc_banner_top vs plc_banner_bottom).
  • Auto-Refresh Rate: Set unit refresh intervals to a minimum of 30 seconds (`30000ms`) to optimize viewability scores and prevent edge rate limits.
