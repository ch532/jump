Connectgold Ads
Developer Documentation & Edge API Integration Guide

1. Architecture Overview

Connectgold Edge Ad Network delivers AI-optimized programmatic ads directly at Cloudflare edge nodes. Real-time contextual decisions are routed via Cloudflare Workers and Workers AI Gateway to maximize yield across direct advertisers and global S2S OpenRTB demand sources (InMobi, ExoClick, Adcash, Opera Ads, PubMatic, Magnite, Adform, etc.).

Core Request Parameters

Parameter Type Description
platform String Target environment: web, webview
format String Ad unit format: banner, native_display, infeed, interstitial_fullscreen, rewarded_video
plc String Unique placement ID on your domain/app (e.g., plc_demo)
pub_id String Your verified Publisher Account ID (e.g., pub_connectgold1234)
consent String User GDPR/CCPA consent state (1 = Granted, 0 = Denied)

2. Publisher Integration Snippets

A. Display Banner Ad Unit (300x250)

Place this container and script in your page HTML where you want the visual banner unit to display.

<!-- Connectgold: Top Banner (300x250) -->
<div id="plc_demo" style="width:300px;min-height:250px;margin:auto;overflow:hidden;"></div>
<script>
(function(){
  function loadAd(){
    var c=(localStorage.getItem("user_ad_consent")==="granted"?"1":"0");
    fetch("https://delicate-night-fb02.trustconnect713.workers.dev/?platform=web&format=banner&plc=plc_demo&pub_id=YOUR_PUBLISHER_ID&consent="+c)
      .then(function(r){return r.json();})
      .then(function(d){
        var b=document.getElementById("plc_demo");if(!b)return;
        var r=d.seatbid&&d.seatbid[0]&&d.seatbid[0].bid&&d.seatbid[0].bid[0];
        if(r&&r.adm){b.innerHTML=r.adm;if(r.nurl)fetch(r.nurl);}
        else if(d.ad){
          var a=d.ad;
          b.innerHTML='<a href="'+a.destination_url+'" target="_blank" rel="noopener"><img src="'+(a.image_url||"https://chyke.online/connectgold_2.png")+'" width="300" height="250" style="border:0;max-width:100%;height:auto;"/></a>';
          if(a.nurl)fetch(a.nurl);
        }
      }).catch(function(e){console.error("Ad error:",e);});
  }
  loadAd();setInterval(loadAd,30000);
})();
</script>

C. Rewarded Video Trigger

Call the function below when a user requests a reward or reaches a game checkpoint.

<!-- Connectgold: Rewarded Video Function -->
<script>
function showRewardedAd(){
  var c=(localStorage.getItem("user_ad_consent")==="granted"?"1":"0");
  fetch("https://delicate-night-fb02.trustconnect713.workers.dev/?platform=web&format=rewarded_video&plc=plc_demo_rewarded&pub_id=YOUR_PUBLISHER_ID&consent="+c)
    .then(function(r){return r.json();})
    .then(function(d){
      if(!d)return alert("No rewarded ad available.");
      var r=d.seatbid&&d.seatbid[0]&&d.seatbid[0].bid&&d.seatbid[0].bid[0];
      if(r&&r.adm){var m=document.createElement("div");m.innerHTML=r.adm;document.body.appendChild(m);if(r.nurl)fetch(r.nurl);}
      else if(d.ad){var a=d.ad;alert("Rewarded Ad: "+(a.title||"Connectgold"));if(a.destination_url)window.open(a.destination_url,"_blank");if(a.nurl)fetch(a.nurl);}
    }).catch(function(e){console.error("Ad error:",e);});
}
</script>

3. Programmatic OpenRTB 2.5 / Prebid Endpoint

Connectgold supports direct Server-to-Server (S2S) OpenRTB 2.5 auctions. External DSPs and Prebid Server bidders can submit standard JSON POST requests to our edge auction pipeline.

Endpoint: POST https://delicate-night-fb02.trustconnect713.workers.dev/openrtb2/auction

Sample OpenRTB 2.5 Request Payload

{
  "id": "cg-req-100234",
  "site": {
    "domain": "chyke.online",
    "page": "https://chyke.online/index.html"
  },
  "imp": [
    {
      "id": "imp-1",
      "banner": { "w": 300, "h": 250 }
    }
  ],
  "user": {
    "ext": { "consent": "CP123456789...TCF_STRING" }
  }
}

4. Multi-Engine Analytics & Telemetry

Ad impression callbacks automatically detect and report impression events to active third-party analytics engines loaded on the host page:

  • Google Analytics (GA4): Dispatches via gtag('event', 'cg_ad_imp', ...)
  • Mixpanel: Tracks event via mixpanel.track('cg_imp', ...)
  • Plausible Analytics: Logs privacy-first goal via plausible('cg_imp', { props: {...} })
